# PV Module Characterization and Analysis

## Overview
This project analyzes the electrical characteristics of a photovoltaic module
using a laboratory PV emulator platform based on the ECO-PVGT experiment setup.

## Objectives
- Obtain I–V characteristics of a PV module
- Obtain P–V characteristics
- Study influence of irradiance variation
- Study influence of temperature variation
- Determine maximum power point (MPP)

## Experimental Setup
Experiments were performed using a photovoltaic emulator platform following
the ECO-PVGT experiment manual and PV emulator user manual.

## Methodology
- Configure PV emulator parameters
- Measure voltage-current characteristics
- Record irradiance-dependent behavior
- Plot I–V and P–V curves
- Analyze maximum power point shift
